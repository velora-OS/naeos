{{ i18n "section_quick_start" }}

{{ i18n "section_quick_start" }} — from spec to code in 4 steps.

1

Install NAEOS

Install via Go, Docker, or download the binary.

bash
go install github.com/NAEOS-foundation/naeos/cmd/naeos@latest
2

Create a Specification

Define your project in YAML or JSON.

yaml
project: my-app
modules:
  - name: auth
    path: ./auth
  - name: api
    path: ./api
    dependencies: [auth]
services:
  - name: gateway
    kind: http
    port: 8080
generation:
  languages: [go, typescript]
3

Run the Pipeline

Execute the full pipeline — parse, validate, generate, and export.

bash
naeos run --input spec.yaml --output-dir ./out
4

Compile for AI Assistance

Generate AI instruction sets for Copilot, Claude, Cursor, and more.

bash
naeos compile --all --input spec.yaml
